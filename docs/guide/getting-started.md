# 开始使用

这里是开源协会的公开知识库，主要收录单片机、ROS、嵌入式、机器人和开源项目相关资料。

## 推荐查找方式

1. 进入 [资料库](/library/) 查看全部资料。
2. 按“单片机”“ROS”“开源项目”等分类浏览。
3. 使用页面顶部搜索框查找关键词，例如 `C语言`、`STM32`、`ROS2`。
4. 打开资料页后，可以在线阅读、预览 PDF，或下载原始文件。

## 当前收录方向

- 培训讲义、实验指导和学习路线
- 单片机外设、传感器、通信协议和项目说明
- ROS 环境配置、功能包使用、仿真和导航资料
- 协会项目复盘、竞赛经验和常见问题

没有收录的主题会在资料库中显示“暂无资料”。
