# 资料分类

资料库按技术方向分类，方便快速浏览。

## 单片机

适合查找 C 语言、嵌入式基础、STM32、51、ESP32、Arduino、外设驱动、传感器和通信协议资料。

当前可在 [资料库](/library/) 的“单片机”分类中查看已收录内容。

## ROS

适合查找 ROS1、ROS2、工作空间配置、节点通信、仿真、导航、SLAM 和机器人项目资料。

如果该分类显示“暂无资料”，表示当前还没有公开收录内容。

## 开源项目

适合查找协会项目文档、项目复盘、竞赛经验、工具链配置和团队实践资料。

如果该分类显示“暂无资料”，表示当前还没有公开收录内容。

## 工具资源

适合查找常用开发工具、调试工具、软件安装和官方文档链接。
