# 常见问题

## 这个方案真的免费吗？

是的。GitHub 公共仓库、GitHub Pages 和 VitePress 都可以免费使用。你暂时不需要购买域名、服务器或数据库。

## 可以做成私密知识库吗？

可以，但复杂度会明显增加。这个版本默认公开只读，所以不要放账号、合同、内部资料或任何敏感信息。

## 可以绑定自己的域名吗？

可以。等你愿意购买域名后，可以在 GitHub Pages 设置中绑定自定义域名。当前版本先使用免费的 `github.io` 地址。

## 可以从飞书同步内容吗？

可以作为后续功能，但第一版先用 Markdown 文件维护，避免一开始就引入 API 权限、同步失败和格式转换等问题。
