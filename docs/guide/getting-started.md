# 开始使用

这个知识库适合公开只读的资料沉淀，例如流程说明、常见问题、工具链接、学习笔记和团队规范。

## 本地预览

安装依赖后运行：

```bash
npm run docs:dev
```

然后打开终端中显示的本地地址。

## 新增文章

1. 在 `docs/guide/` 下创建一个新的 `.md` 文件。
2. 写入标题和正文。
3. 在 `docs/.vitepress/config.ts` 的 `sidebar` 中加入文章链接。
4. 提交到 GitHub 后，线上网站会自动更新。

## 内容建议

每篇文章尽量只解决一个主题。标题要具体，正文优先写清楚“适用场景、操作步骤、注意事项、相关链接”。
