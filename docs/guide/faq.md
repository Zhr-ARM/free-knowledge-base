# 常见问题

## 我把文件放进 uploads 后，为什么网站没变化？

需要先运行：

```bash
npm run sync:uploads
```

如果要更新线上网站，还需要运行：

```bash
npm run docs:deploy
```

## Word 文档能直接显示吗？

`.docx` 会尽量转换为网页内容，并保留原文件下载链接。旧版 `.doc` 浏览器不能直接显示，系统会生成下载页；建议先另存为 `.docx` 或 `.pdf`。

## PDF 会怎么显示？

PDF 会生成一个资料页面，并在页面中嵌入预览框，同时提供打开或下载链接。

## 可以上传哪些主题？

建议优先上传单片机、ROS、机器人、嵌入式、开源项目、培训资料和竞赛经验相关内容。
