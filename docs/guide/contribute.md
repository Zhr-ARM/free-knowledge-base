# 查找资料

知识库的资料会按主题分类展示。你可以从导航栏进入 [资料库](/library/)，也可以直接打开对应栏目。

## 按分类浏览

- [单片机](/guide/mcu)：适合查找 C 语言、STM32、51、ESP32、Arduino、传感器和外设资料。
- [ROS](/guide/ros)：适合查找 ROS1、ROS2、仿真、导航、SLAM 和机器人项目资料。
- [工具资源](/guide/resources)：适合查找常用软件、调试工具和参考链接。

## 使用搜索

页面顶部有搜索框。输入关键词后，可以快速定位标题和正文中包含相关内容的资料。

推荐关键词示例：

- `C语言`
- `STM32`
- `GPIO`
- `ROS2`
- `SLAM`
- `导航`

## 资料状态

如果某个分类还没有资料，页面会显示“暂无资料”。这表示当前知识库还没有收录对应内容，不代表协会没有相关方向。
