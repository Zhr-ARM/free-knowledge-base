# 开始使用

这里是开源协会的公开知识库，用来沉淀单片机、ROS、嵌入式、机器人和开源项目相关资料。

## 适合放什么

- 培训讲义、实验指导和学习路线
- 单片机外设、传感器、通信协议和项目代码说明
- ROS 环境配置、功能包使用、仿真和导航资料
- 协会项目复盘、竞赛经验、工具链配置和常见问题

## 查看资料

进入 [资料库](/library/) 可以看到从 `uploads/` 自动生成的资料列表。页面支持站内搜索，适合快速查找关键词。

## 新增资料

把 `.md`、`.pdf`、`.docx`、`.doc` 文件放入项目根目录的 `uploads/` 文件夹，然后运行：

```bash
npm run sync:uploads
```

确认本地展示正常后，再发布：

```bash
npm run docs:deploy
```
