# 上传资料说明

把要展示在网站里的资料放在这个文件夹里。

推荐目录：

- `单片机/`：STM32、51、ESP32、Arduino、传感器、外设驱动等资料
- `ROS/`：ROS1、ROS2、机器人建模、导航、仿真、SLAM 等资料
- `开源项目/`：协会项目文档、培训资料、竞赛记录、经验总结等资料

支持格式：

- `.md`：会转换为知识库页面
- `.pdf`：会生成网页预览页
- `.docx`：会尽量转换为网页内容，并保留原文件下载链接
- `.doc`：会生成下载页；建议转成 `.docx` 或 `.pdf` 后再上传

放入文件后，在项目根目录运行：

```bash
npm run sync:uploads
```

如果要发布到线上，再运行：

```bash
npm run docs:deploy
```
